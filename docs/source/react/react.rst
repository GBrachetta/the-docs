React
=====

Basics
######

Functional Components
*********************

.. code-block:: jsx

  import React from 'react'

  const MyComponent = () => {
    return (
      <div></div>
    )
  }

  export default MyComponent

Stateful Components
*******************

.. code-block:: jsx

  import React, { Component } from 'react';

  class MyClassComponent extends Component {
    render() {
      return <div />;
    }
  }

  export default MyClassComponent;
